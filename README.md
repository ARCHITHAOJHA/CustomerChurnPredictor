# Customer Churn Prediction

Simple Streamlit frontend for predicting customer churn using pre-trained models.

Requirements

- Python 3.8+
- See `requirements.txt` for packages (install with `pip install -r requirements.txt`).

Run locally

```bash
cd app
streamlit run app.py
```

Notes

- The app expects the model files in `models/` (already included in this repo).
- To tweak visuals, edit `app/app.py`.
